# Projeto Git e GitHub - Exercício EBAC

Este repositório foi criado para demonstrar o uso do **Git** e **GitHub** no controle de versões, com foco em práticas de desenvolvimento no contexto de **Back-End**.

## Autor

- **José Zaltar**
- **E-mail**: [josezaltar@gmail.com]

## Descrição

O projeto aborda o uso do **Git** para gerenciamento de código-fonte, controle de versões e colaboração no desenvolvimento de software. O repositório foi criado como parte de um exercício prático na **EBAC** para reforçar o aprendizado do uso de **branches**, **commits**, **pull requests** e outras funcionalidades do **Git** e **GitHub**.

### Funcionalidades

- Criação de **branches** para organização de novas features.
- **Commits** descritivos para controle de versão.
- **Pull requests** para integração de diferentes branches de desenvolvimento.

## Como rodar o projeto

1. Clone o repositório:

   ```bash
   git clone https://github.com/josezaltar/git-exericio-ebac.git
   ```

2. Acesse o diretório do projeto:

   ```bash
   cd git-exericio-ebac
   ```

3. Execute os scripts conforme necessário.
